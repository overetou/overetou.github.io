Malloc (C)

  • Home

Reproduction of the C standard memory allocation function.

This project introduces the way C programs manage dynamically allocated memory. The task consists in learning the fundamental system calls that allocate blocs of dynamic memory and creating a system able to distribute chunks of those blocs and manage them. To do so, it is imperative to understand how the computer processes raw memory.

To test a malloc implementation, it is possible to use it instead of the original in an already compiled program. I did not reproduce all functions of the alloc family so complex programs are bound to crash but a lot of command-line interface programs do work with my versions, such as ls, htop, git, vim and others.

All examples above are using my implementation of malloc.
From left to right and top to bottom: the "ls" command launched on the root of the project; vim (a text editor in command line); "htop" (an interactive system monitor in command line) and "man", the Unix command line manual, displaying the page of malloc.
