Garden of Mists (Python3 + libgit + gtk3)

  • Home

Desktop application dedicated to creative writing.

The creation of this software was motivated by the realization that as a text grows, it becomes longer to find precise parts of it. The project attempts to solve this problem.

Non exhaustive list of functionalies presented in the above video:
  • Project management in a welcome menu.
  • Text edition assisted by insertion buttons. Keyboard shortcuts can be used in their stead.
  • File management inside a project.
  • Temporal tags insertion and management in the text. The user can select a date and a label. When he clicks on a tag in the interface, the text cursor relocates itself where he was when the tag was created. Thanks to an in-house algorithm, the position of tags in the text tends to stay relevant no matter what modifications are made.
  • Dark and light mode. This parameter, as well as others, is preserved from a session to another.

But there is still more.

The application integrates an intuitive management of Git. It is possible to save the state of a project and navigate between each of its version with a few clicks.
Git branches are also supported. That way, if the user wants to modify an old version of his project, the application automaticaly creates a new branch. Each version of a project is sorted by time of creation and has a label.

Finally, a priority for the project was to offer a simple interface. The user is free to choose the background for each visual mode (light and dark). He can also hide one or both of the side panels to minimize visual distractions.
On a side note the application is made to be used in fullscreen but my capture methods only allowed me to film it in windowed mode.

The software was almost finished, but put in standby when I realized that the packaging of the app was failing on windows because of a conflict in the libraries. This allowed me to learn that before any actual progress is made in a project, all tools should be tested together for release on each supported platform. The project is not currently abandoned but it will come back in a different form.
