Cryptographie.

Le but du projet est de reproduire les algorithmes de hash md5 et sha256. Dans l'image de dessous, mon programme (./ft_ssl) et la commande de hash de linux (sha256sum) calculent tous deux l'empreinte sha256 du fichier "file.txt". On peut constater qu'elles sont identiques.

Le programme supporte toute sorte de fichier, y compris les binaires et les fichiers de plusieurs gigas. Dans l'exemple suivant, md5 est lancé sur l'exécutable "ft_ssl". Comme avant, on lance notre implémentation puis celle de Linux afin de pouvoir comparer les résultats.

Le dernier exemple montre que le hashing de fichiers volumineux est exact: On lance les programmes de hash sur une image de système d'exploitation. Les résultats sont identiques. La dernière commande, "du", affiche le poids du fichier, soit 1.9 Go.
