Packet transmission through the internet.

Ping is a command that allows measuring the quality of the transmission in a network, as well as its characteristics. The goal of the project is to recreate the essential functionalities of the program. In the example below, I successively launched my implementation and the official Linux one. We can see that the reported response times are similar.

This project is a good introduction to the network programming concepts because it requires learning to format, receive and send data packets at a low level between distant machines.
