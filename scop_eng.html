Scop (C + OpenGL)

Affichage d'objet 3D.

Scop is a 3D object viewing program created with OpenGl in C. SDL2 was used to generate the context, the windowing and to take user inputs.
The program reads ".obj" files (a standard to store 3D data) and .bmp files (a standard for pictures). It is able to apply the content of the bmp file as a texture on the surface of the 3D object. It also supports a wireframe mode to visualize the modelisation triangles.

The program uses its own shaders and parses every file without the help of any external library. All controls are exclusively available through the keyboard. It is possible to move the object, to rotate it in different ways and to zoom / dezoom.
