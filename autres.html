Lem-in

Le projet consiste à guider des fourmis d'une alcove A à une alcove B en empruntant le chemin le plus court possible au sein d'une fourmilière, le tout sans employer trop de temps à calculer l'itinéraire. Pour ce faire, il est courant d'employer un algorithme de parcour en largeur. Projet réalisé en binôme.

Corewar

Corewar consiste à créer une machine virtuelle sous la forme d'une arène qui éxécute du code asm. Les champions qui combattent dans cette arène sont des fragments de code assembleur et ont pour but d'écraser ou de corrompre les champions adverses. Il est également demandé de créer un convertisseur de code assembleur vers un format binaire interprétable par l'arêne. Projet réalisé en équipe de 4. En bonus, nous avons réalisé un désassembleur, soit un executable qui crée un fichier de code source asm à partir d'un fichier binaire généré auparavant.

Snowcrash

Snowcrash est un projet d'introduction à la sécurité. Il prend la forme d'une image de système d'exploitation (que l'on lance dans une machine virtuelle) dans lequel le challenger doit parvenir à obtenir des privilèges réservés à d'autres utilisateurs. On utilise pour cela différentes failles classiques de sécurité.
Parmis elles: le dechiffrage de hash de mot de passe stocké publiquement, l'analyse de flux TCP, l'exploitation de tâches cron, l'injection de code par différents moyens (variables d'environement, fonctions backend vulnérables, debugger, ...), les "race conditions" et la modification du path pour singer une fonction.

Push-Swap

Projet centré sur l'optimisation. On a deux piles de nombres et un jeu d'instruction limité pour interchanger les nombres qui y sont stockés. Le but est de fournir un nombre minimum d'instruction permettant de trier ces nombres par ordre de grandeur. Le projet est jugé en fonction de ce nombre d'instructions. C'est une bonne introduction aux différents algorithmes de tri, notamment le quick sort, et à la problématique de leur adaptation dans un cadre différent de ceux dans lesquels ils sont typiquement mis en place.
